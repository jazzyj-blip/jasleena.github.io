Cross-Section of Earth's Mantle

An Interactive Data Visualization

Project Description

In this project, I aim to create an interactive data visualization that explores the internal structure of the Earth mantle in a way that is both scientifically accurate and engaging for users.

The visualization will focus on the main subdivisions of the mantle:

  • Upper Mantle
  • Transition Zone
  • Lower Mantle

Each of these layers will be represented with their relative depths, thicknesses, and key physical characteristics such as temperature ranges, composition, and density.

The interactive component of the visualization will provide:

  • A scalable cross-section` with clearly labeled layers.
  • A depth scale to contextualize the information.
  • Quick facts that can be revealed through hover or click interactions.
  • A depth slider that allows users to highlight specific depths of interest dynamically.

The overall goal of the project is to communicate complex geophysical information in a clear, accessible, and visually compelling way. By using interactive elements, users will be able to:

  • Compare the relative thicknesses of different mantle layers.
  • Understand and visualize how physical properties change with depth.
  • Quickly interpret data without needing prior geoscience knowledge.

This project is especially suitable for a data visualization course because it combines several important elements:

  • Accurate domain data from earth sciences.
  • Clean visual encoding to make abstract concepts tangible.
  • Accessible interaction design that supports exploration and learning.
  • Reproducible code with a short and explainable structure, making it easy for others to adapt or extend.
